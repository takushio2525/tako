# tako アプリアイコン

モチーフ「タコ + ターミナル + AI」の方向性違い 4 案からの選定。すべて完全オリジナルの
手描き SVG（1024×1024 viewBox、macOS 作法の角丸正方形 rx=234 ベース、
フラット〜軽いグラデーション）。

## 選定結果（2026-06-11 確定）

- **採用: A 案（`icon-a.svg`）** — ダークターミナル背景 + コーラルピンクのタコ +
  緑のプロンプト `>_` の王道マスコット案。Dock 最小サイズでの判別性とコンセプトの
  伝わりやすさで選定
- **没: B〜D 案** — 比較記録としてファイルは保存したまま残す（削除しない）。
  再選定やバリエーション展開（ベータ版アイコン等）の素材として参照可

`.icns` 化とアプリへの組み込みは `scripts/build-app.sh` が行う（A 案固定。
rsvg-convert があれば SVG から全サイズ描画、無ければ `preview/icon-a-1024.png` から
sips で縮小生成）。

## 各案のコンセプト

| ファイル | コンセプト（1 行） |
|---|---|
| `icon-a.svg` | ダークターミナルを背景に、コーラルピンクのタコと緑のプロンプト `>_` を縦に並べた王道マスコット案 |
| `icon-b.svg` | タコの頭そのものをターミナルウィンドウ（信号ドット + ブロックカーソルの目）にし、下から触手が生える擬生物化案 |
| `icon-c.svg` | 触手 1 本（吸盤付き・先端カール）でプロンプト記号「>」を描き、カーソル矩形と AI ノード網を添えた抽象ロゴ案 |
| `icon-d.svg` | ターミナルのブロックカーソルそのものがタコになり、プロンプト「>」の隣に棲んでいる超ミニマル案 |

## プレビュー

`preview/` に各案の PNG を同梱（`icon-{a..d}-1024.png` が原寸、`icon-{a..d}-64.png` が
Dock 最小級サイズでの判別性確認用）。

## 再生成手順

書き出しには librsvg を使う（`brew install librsvg`）。

```sh
cd assets/icon
for v in a b c d; do
  rsvg-convert -w 1024 -h 1024 icon-$v.svg -o preview/icon-$v-1024.png
  rsvg-convert -w 64  -h 64  icon-$v.svg -o preview/icon-$v-64.png
done
```
