# 言語設定

## 必須概念

- 回答・コミットメッセージ・コードコメントの記述言語を明示する
- 技術用語やコード識別子は原語のまま維持する旨を記載する

## 参考テンプレート

### 日本語ユーザー

```markdown
## 言語

- 回答・コミットメッセージ・ドキュメントはすべて**日本語**で記述する
- コードのコメントも日本語
- 技術用語・コード識別子は原語のまま維持する
```

### English User

```markdown
## Language

- All responses, commit messages, and documentation in **English**
- Code comments in English
- Technical terms and code identifiers remain in their original form
```
