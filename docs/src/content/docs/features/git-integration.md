tako の右サイドバーには git ビューが組み込まれています。現在の作業ディレクトリの git 情報をリアルタイムで表示します。

## git ビュー

ステータスバーの git ボタンで右サイドバーの git ビューを開閉できます。4 つのセクションがアコーディオン形式で並びます。

### ブランチ情報

現在のブランチ名と最新コミットのハッシュを表示します。

### 変更ファイル一覧

`git status` に相当する変更ファイルの一覧。ステージ済み・未ステージ・未追跡を色分けして表示します。

### コミットグラフ

`git log` のコミット履歴をグラフ表示します。コミットをクリックするとそのコミットの diff が表示されます。

### diff 表示

選択したコミットまたはファイルの差分を表示します。追加行は緑、削除行は赤でハイライトされます。

## 自動更新

git ビューは 2 秒間隔のポーリングで自動更新されます。パネルを開いた瞬間にも即時 fetch が走るため、常に最新の状態が反映されます。

作業ディレクトリ（cwd）の変更にも追従します。タブ内のフォーカスペインが別のリポジトリに移動すれば、git ビューもそのリポジトリの情報に切り替わります。

## CLI / MCP

```bash
# git ログを取得
tako git log

# git diff を取得
tako git diff
```

MCP ツール `tako_git_log` / `tako_git_diff` からも同じ情報を取得できます。AI が「現在のブランチのコミット履歴を確認して」といった操作を自律的に行えます。
