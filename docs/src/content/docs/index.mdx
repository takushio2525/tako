## なぜ tako？

通常のターミナルで Claude Code を使うと、子エージェントや dev サーバーが別タブ・別ウィンドウに散らばり、AI が何をしているのか見失いがちです。

tako は**AI エージェントが自分の作業環境を自律的に組み立てる**ことを前提に設計されたターミナルです。内蔵 MCP サーバーにより、AI は設定ゼロで画面分割・コマンド実行・ファイル表示・ポート検知を操作できます。人間はタブを眺めるだけで、複数エージェントの進捗と成果物を一目で把握できます。

## はじめよう

```bash
# 1. GitHub Releases からダウンロード＆インストール
# 2. MCP サーバーを登録（初回のみ）
tako setup-mcp

# 3. tako 内で Claude Code を起動 — もう AI は tako を使える
claude
```

詳しい手順は [セットアップガイド](/getting-started/) をご覧ください。
