<section class="tako-section">
<h2 class="tako-section-title"><span class="tako-section-dash">—</span> tako でできること</h2>

<div class="tako-card-grid">
<div class="tako-card">
<div class="tako-card-icon tako-card-icon-pink">⚡</div>
<h3>内蔵 MCP サーバー</h3>
<p>tako の中で Claude Code を起動すれば、AI が設定ゼロでペインの分割・コマンド実行・ファイル表示を行える。「隣のペインで dev サーバー起動して」と言うだけ。50 以上の MCP ツールを内蔵。</p>
</div>

<div class="tako-card">
<div class="tako-card-icon tako-card-icon-pink">⊞</div>
<h3>タブ＆ペイン管理</h3>
<p>iTerm2 ライクな分割レイアウトにドラッグ＆ドロップ対応。エージェント・dev サーバー・ログを「1 タブ = 1 作業グループ」にまとめて、どのプロジェクトで何が動いているかが一目でわかる。</p>
</div>

<div class="tako-card">
<div class="tako-card-icon tako-card-icon-green">↻</div>
<h3>再起動しても消えない</h3>
<p>tmux バックエンドにより、tako を閉じて再起動しても実行中のプロセスと画面内容がそのまま復元される。エージェントの長時間タスクも、dev サーバーもそのまま。</p>
</div>

<div class="tako-card">
<div class="tako-card-icon tako-card-icon-green">◇</div>
<h3>ファイルプレビュー</h3>
<p>AI が「このファイル見て」と言ったら、シンタックスハイライト付きのコードプレビューや Markdown レンダリングがペイン内に直接表示される。ターミナルから出ずにレビューできる。</p>
</div>

<div class="tako-card">
<div class="tako-card-icon tako-card-icon-pink">♪</div>
<h3>オーケストレーション</h3>
<p><code>tako master</code> で司令塔の AI を起動すれば、複数の子エージェントが同じタブのペインに並んで働く。依頼は自然言語で 1 回だけ。監視も回収も自動。<a href="/features/orchestration/">詳しく →</a></p>
</div>

<div class="tako-card">
<div class="tako-card-icon tako-card-icon-green">📱</div>
<h3>スマホからリモート監視</h3>
<p><code>tako remote start</code> で QR コードを表示、スマホで読み取ればどこからでもターミナルの様子を確認できる。エージェントに長時間タスクを任せて外出しても安心。</p>
</div>
</div>
</section>
