<section class="tako-section">
<h2 class="tako-section-title"><span class="tako-section-num">01</span> tako でできること</h2>

<div class="tako-card-grid">
<div class="tako-card">
<div class="tako-card-icon tako-card-icon-pink">⚡</div>
<h3>内蔵 MCP サーバー</h3>
<p>tako の中で Claude Code を起動すれば、AI は設定ゼロで自動的にペインの分割・コマンド実行・ファイル表示ができる。「隣のペインで dev サーバー起動して」が一言で通る。</p>
</div>

<div class="tako-card">
<div class="tako-card-icon tako-card-icon-pink">⊞</div>
<h3>タブ＆ペイン管理</h3>
<p>iTerm2 ライクな分割レイアウトにドラッグ＆ドロップ対応。AI エージェント・dev サーバー・ログを「1 タブ = 1 作業グループ」にまとめて全体像を把握できる。</p>
</div>

<div class="tako-card">
<div class="tako-card-icon tako-card-icon-green">↻</div>
<h3>再起動しても消えない</h3>
<p>tmux バックエンドにより、tako を閉じて再起動しても実行中のプロセスと画面内容がそのまま復元される。エージェントの長時間タスクも安心。</p>
</div>

<div class="tako-card">
<div class="tako-card-icon tako-card-icon-green">◇</div>
<h3>ファイルプレビュー</h3>
<p>AI が「このファイル見て」と言ったら、シンタックスハイライト付きのコードプレビューや Markdown レンダリングがペイン内に直接表示される。</p>
</div>
</div>
</section>
